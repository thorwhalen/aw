aw.tools

Tools for executing code and other agent actions.

Provides code execution capabilities with safe defaults and extensibility for robust backends like LangChain’s CodeInterpreter.

class aw.tools.CodeInterpreterTool(allowed_modules: list[str] | None = None, global_context: dict | None = None, executor: Callable[[str, dict], ExecutionResult] | None = None)[source]

Tool for executing Python code in a controlled environment.

Provides a safe default implementation using exec() with limited namespace, and allows injection of more robust backends.

Example

>>> tool = CodeInterpreterTool()
>>> result = tool("x = 5; y = x * 2; print(y)")
>>> result.success
True
>>> result.output.strip()
'10'
